Detailed Description

Namespace for dynamic programming classes.

Many of theses classes use hard-to-read C++ template specializations, because we are trying to avoid the overhead of virtual functions and and branching logic in the time-critical inner loop of dynamic programming.

Whether alignment operands have preexisting gaps (e.g. alignments) or not (e.g. single sequences)

Enumerator:
DP_ALIGN_UNGAPPED_SEQUENCES 

Alignment with no preexisting gaps can be faster and truly optimal.

DP_ALIGN_GAPPED_ALIGNMENTS 

Alignment of pregapped sequences is slower and heuristic (problem is NP-hard)
