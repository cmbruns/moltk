Detailed Description

template<class U, class Y = Real>
struct moltk::units::Quantity< U, Y >

Quantity represents a physical value with a unit, such as "5.6 nanometers".

For efficiency, Quantity<> should compile to a double in C++. This means: no virtual methods no data members other than "value" unit is a class with a typedef These restrictions need not apply to unit class, which should do whatever it needs to, to be wrapped conveniently in python.
