Module _moltk'

The _moltk module provides tools for protein sequence/structure alignment and visualization.

This submodule "moltk._moltk" represents the binary extension implemented in C++. The top level module "moltk" contains all of the symbols from "moltk._moltk", so use "import moltk" of "from moltk import <whatever>" to get symbols from "moltk._moltk".
