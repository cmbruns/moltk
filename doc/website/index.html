What is MolTK?

MolTK is free software for analysis of molecular structures. MolTK was created by Christopher Bruns.

There are several aspects of MolTK; in order from easiest-to-use to most-powerful:

  • A computer application for viewing and aligning molecular sequences and structures.
  • An interactive Python programming language module for aligning and otherwise computing on molecular sequences and structures.
  • A C++ API for efficient compution with molecular sequences and structures.
  • An open source project so you can modify and extend MolTK yourself.
